Website for UCL's Technology Society from 2015 to 2016.

Source code Preview

From October 2015 until June 2016, I was the society's "webmaster" and I built and maintained the website.

The website was built in Jekyll and ran on GitHub Pages. My favourite features were the ones that were kind of hacked together and uncommon for Jekyll sites, like:

  • Calendar subscriptions
  • JSON feeds
  • Facebook event synchonisation
