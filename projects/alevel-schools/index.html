A website for comparing A-level schools by subject performance.

Source code Website

The Department for Education publishes yearly school performance data, which apart from overall performance also includes subject-specific performance for A-levels. This website tries to expose and allow comparison by this data, which wasn't shown by most of the school comparison websites at the time.
